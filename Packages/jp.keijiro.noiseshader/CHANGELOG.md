# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](http://semver.org/spec/v2.0.0.html).

## [3.0.0] - 2025-01-26

### Added

- Added a 1D gradient noise function (`GradientNoise`).

### Changed

- Pulled updates from the original webgl-noise repository.
- Updated the normalization factor for the classic noise functions.

## [2.0.1] - 2025-01-22

### Changed

- Pulled an update from the original webgl-noise repository to adjust a constant in the permutation function, slightly improving randomness.
